Classes that enable an external agent to trade with the market. {@link net.commerce.zocalo.rpc.RPCServer} provides an interface via xmlrpc that relies on {@link net.commerce.zocalo.rpc.RPCTrade}, which uses {@link net.commerce.zocalo.JspSupport.TradeSupport}.
